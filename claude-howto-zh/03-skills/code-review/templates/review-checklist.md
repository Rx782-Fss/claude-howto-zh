# 代码审查检查清单

## 安全检查清单
- [ ] 无硬编码凭证或密钥信息
- [ ] 所有用户输入均已做校验处理
- [ ] SQL 注入防护（使用参数化查询）
- [ ] 状态变更操作的 CSRF 防护
- [ ] XSS 防护（正确的转义处理）
- [ ] 受保护端点的身份认证检查
- [ ] 资源的授权检查
- [ ] 安全的密码哈希（bcrypt、argon2）
- [ ] 日志中无敏感数据
- [ ] 强制使用 HTTPS

## 性能检查清单
- [ ] 无 N+1 查询问题
- [ ] 合理使用了索引
- [ ] 在有收益的地方实现了缓存
- [ ] 主线程上无阻塞操作
- [ ] 正确使用了 async/await
- [ ] 大数据集做了分页处理
- [ ] 数据库连接使用连接池
- [ ] 正则表达式已优化
- [ ] 无不必要的对象创建
- [ ] 已防止内存泄漏

## 质量检查清单
- [ ] 函数长度不超过 50 行
- [ ] 变量命名清晰明确
- [ ] 无重复代码
- [ ] 有适当的错误处理
- [ ] 注释解释的是"为什么"，而不是"是什么"
- [ ] 生产环境代码中无 console.log
- [ ] 类型检查（TypeScript / JSDoc）
- [ ] 遵循 SOLID 原则
- [ ] 设计模式正确应用
- [ ] 代码自文档化

## 测试检查清单
- [ ] 已编写单元测试
- [ ] 边界情况已覆盖
- [ ] 错误场景已测试
- [ ] 存在集成测试
- [ ] 覆盖率 > 80%
- [ ] 无不稳定的测试
- [ ] Mock 了外部依赖
- [ ] 测试名称清晰易懂
