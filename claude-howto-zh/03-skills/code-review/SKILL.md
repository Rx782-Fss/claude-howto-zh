# Code Review（代码审查）技能

本技能提供全面的代码审查能力，重点关注：

1. **安全分析**
   - 身份认证/授权问题
   - 数据泄露风险
   - 注入类漏洞
   - 加密弱点
   - 敏感数据日志记录

2. **性能审查**
   - 算法效率分析（Big O 分析）
   - 内存优化
   - 数据库查询优化
   - 缓存机会
   - 并发问题

3. **代码质量**
   - SOLID 原则
   - 设计模式
   - 命名规范
   - 文档完整性
   - 测试覆盖率

4. **可维护性**
   - 代码可读性
   - 函数长度（应 < 50 行）
   - 圈复杂度
   - 依赖管理
   - 类型安全

## 审查模板

对每段被审查的代码，提供：

### 摘要
- 整体质量评估（1-5 分）
- 关键发现数量
- 建议优先处理领域

### 关键问题（如有）
- **问题描述**：清晰描述
- **位置**：文件和行号
- **影响**：为什么重要
- **严重程度**：Critical（严重）/ High（高）/ Medium（中）
- **修复建议**：代码示例

### 分类发现

#### 安全问题（如有）
列出安全漏洞及示例

#### 性能问题（如有）
列出性能问题及复杂度分析

#### 质量问题（如有）
列出代码质量问题及重构建议

#### 可维护性问题（如有）
列出可维护性问题及改进方案
