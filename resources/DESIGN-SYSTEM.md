# Claude How To - 设计系统

## 视觉标识

### 图标设计概念:指南针配代码括号

Claude How To 图标使用**指南针配合 `>` 代码括号**来表示代码导航指引:

```
     N (绿色)
     ▲
     │
W ───>─── E     指南针 = 引导/方向
     │          > 括号 = 代码/终端/CLI
     ▼
     S (黑色)
```

这创造了:
- **视觉清晰性**:立即传达"代码导航指南"
- **象征意义**:指南针 = 找到方向;`>` = 代码/终端
- **可扩展性**:从 16px 到 512px 的任意尺寸均适用
- **品牌一致性**:匹配开发者工具美学风格,极简配色

## 色彩体系

### 调色板

| 颜色 | 十六进制 | RGB | 用途 |
|-------|---------|-----|------|
| 黑色(主色) | `#000000` | 0, 0, 0 | 主笔画、文本、南针 |
| 白色(背景) | `#FFFFFF` | 255, 255, 255 | 浅色背景 |
| 灰色(辅色) | `#6B7280` | 107, 114, 128 | 次要刻度、次要文本 |
| 亮绿(强调色) | `#22C55E` | 34, 197, 94 | 北针、中心点、强调线 |
| 近黑(深色背景) | `#0A0A0A` | 10, 10, 10 | 暗色模式背景 |

### 对比度(WCAG 标准)

- 黑底白字:**21:1** AAA 级
- 灰底白字:**4.6:1** AA 级
- 绿底白字:**3.2:1**(仅装饰用途,不用于文字)
- 白底暗色:**19.5:1** AAA 级

### 强调色规则

**亮绿色(#22C55E)仅用于高亮:**
- 指南针北针
- 中心圆点
- 强调下划线/边框
- 绝不作为背景色
- 绝不用于正文

## 图标规格

### 指南针规格

```
元素                 | 描边/填充      | 颜色
---------------------|---------------|------------------
外环                  | 3px 描边      | 黑色 / 白色(暗色模式)
北向刻度              | 2.5px 描边    | 黑色 / 白色(暗色模式)
其他基本方位刻度      | 2px 描边      | 灰色 / 白色 50%(暗色模式)
中间方位刻度          | 1.5px 描边      | 灰色 / 白色 40%(暗色模式)
北针                   | 填充多边形     | #22C55E(始终为绿色)
南针                   | 填充多边形     | 黑色 / 白色(暗色模式)
> 括号                | 3px 描边      | 黑色 / 白色(暗色模式)
中心点                 | 填充圆形       | #22C55E(始终为绿色)
```

### 尺寸递进

```
16px  → 仅环 + 针 + V 形(极简版)
32px  → 添加基本方位刻度
64px  → 添加中间方位刻度
128px → 完整细节,所有元素清晰
256px → 最大细节,加粗描边
```

## 可访问性

### 色彩对比度
- 所有文字满足 WCAG AA(最低 4.5:1)
- 绿色强调色为装饰性,非信息性

### 可扩展性
- 矢量格式确保任意尺寸下清晰
- 几何形状在 16px 下仍可识别
- 根据可用尺寸渐进式增加细节
