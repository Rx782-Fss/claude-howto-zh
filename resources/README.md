# Claude How To - 品牌资源

完整的 Logo、图标和 Favicon 资源集合。所有资源采用 V3.0 设计:指南针配合代码括号(`>`)符号,代表在代码中的导航指引----使用黑白灰配色方案搭配亮绿色(#22C55E)强调色。

## 设计概念(V3.0)

**指南针配代码括号** -- 引导遇见代码:
- **指南针环** = 导航、找到方向
- **绿色北针** = 方向、学习路径上的进步
- **黑色南针** = 扎根、稳固基础
- **`>` 括号** = 终端提示符、代码、CLI 上下文
- **刻度标记** = 精准、结构化学习

## 色彩规范

| 颜色 | 十六进制 | 用途 |
|------|---------|------|
| 黑色(主色) | `#000000` | 主笔画、文本、南针 |
| 白色(背景) | `#FFFFFF` | 浅色背景 |
| 灰色(辅色) | `#6B7280` | 次要刻度、次要文本 |
| 亮绿(强调色) | `#22C55E` | 北针、中心点、强调线 |

**强调色规则**:亮绿色仅用于高亮元素(针尖、圆点、强调线),绝不作为背景或正文颜色使用。

## 使用指南

- 在白色或深色(#0A0A0A)背景上使用
- 保持比例缩放
- Logo 周围保留至少 logo 高度一半的留白空间
- 根据背景选择浅色/深色变体
